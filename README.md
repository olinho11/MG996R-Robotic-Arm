# MG996R-Robotic-Arm
An MG996R Robotic Arm designed completely from scratch by me. (code, CAD, hardware)
Overview
Before jumping straight into the full MG996R build, I'm designing a prototype arm using SG90 servo motors to get a feel for the design process. Things like joint geometry, servo mounting, friction fit tolerances, cable routing, and how all the parts interact in real life. This prototype is essential for understanding what works and what doesn't before committing to larger, more powerful hardware.
The full arm will be built with MG996R servos, which offer significantly more torque and precision than the SG90, making it suitable for a real functional robotic arm.

# Planned Features

Multi-DOF arm designed from scratch in Onshape
Custom 3D printed parts
Arduino-based control system
Custom-written code for movement and control
